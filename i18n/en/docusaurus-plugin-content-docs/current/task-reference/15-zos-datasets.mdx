# `zos-datasets` — MVS datasets and USS files

Operations on z/OS MVS datasets (sequential and partitioned PDS/PDSE) **and USS files** (Unix System Services): upload, download, create, delete, list, chmod, copy, rename.

```yaml
- uses: zos-datasets
  with:
    action: upload
    localPath: ./src/cobol
    dataset: USER.MYAPP.SRC
    lpar: dev01
```

## Common inputs

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `action` | string | yes | `upload`, `download`, `create`, `delete`, `list`, `exists`, `copy`, `rename`, `mkdir`, `chmod`. |
| `lpar` | string | no | Target LPAR profile (default: active profile). |

**MVS vs USS discrimination:** if `ussPath` is present → USS target. If `dataset` is present → MVS dataset. `mkdir` and `chmod` are USS-only.

## MVS actions

### `upload`

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `dataset` | string | yes | Target dataset. For a PDS, each local file becomes a member. |
| `localPath` | string | yes | Local file or directory. |
| `encoding` | string | no | Source codepage. Empty = automatic EBCDIC conversion. |
| `binary` | boolean | no | `true` = binary transfer without conversion. Default `false`. |

**Member naming**: extension stripped → uppercase → invalid characters removed → truncated to 8 → `A` prefix if first character is a digit.

### `download`

| Field | Type | Required |
|-------|------|----------|
| `dataset` | string | yes — dataset or member (`USER.SRC(MEMBER)`) |
| `localPath` | string | yes — local destination |
| `encoding` | string | no |
| `binary` | boolean | no |

### `create`

| Field | Type | Required | Default | Description |
|-------|------|----------|---------|-------------|
| `dataset` | string | yes | — | Dataset name (max 44 chars). |
| `datasetType` | string | no | `seq` | `seq`, `pds`, or `pdse`. |
| `recfm` | string | no | `FB` | `FB`, `VB`, `F`, `U`, `FBA`, `VBA`. |
| `lrecl` | integer | no | `80` | Record length. |
| `primary` | integer | no | `1` | Primary allocation in tracks. |
| `secondary` | integer | no | `1` | Secondary allocation in tracks. |
| `dirblk` | integer | no | — | Directory blocks (for PDS). |

### `delete`

| Field | Type | Required |
|-------|------|----------|
| `dataset` | string | yes — dataset or `PDS(MEMBER)` |

### `list`

| Field | Type | Required |
|-------|------|----------|
| `pattern` | string | one of the two — ISPF pattern (`USER.MYAPP.*`) |
| `membersOf` | string | one of the two — list members of this PDS |

### `copy` / `rename`

| Field | Type | Required |
|-------|------|----------|
| `source` | string | yes |
| `target` | string | yes |

### `exists`

| Field | Type | Required |
|-------|------|----------|
| `dataset` | string | yes — dataset, member (`USER.PDS(MBR)`) |

## USS actions

### `upload` / `download`

| Field | Type | Required |
|-------|------|----------|
| `ussPath` | string | yes — absolute USS path (starts with `/`) |
| `localPath` | string | yes |
| `encoding` | string | no |
| `binary` | boolean | no |

### `mkdir`

| Field | Type | Required | Default |
|-------|------|----------|---------|
| `ussPath` | string | yes | — |
| `recursive` | boolean | no | `true` |
| `mode` | string | no | `0755` |

### `delete` (USS)

| Field | Type | Required | Default |
|-------|------|----------|---------|
| `ussPath` | string | yes | — |
| `recursive` | boolean | no | `false` |

### `exists` (USS)

| Field | Type | Required |
|-------|------|----------|
| `ussPath` | string | yes |

### `chmod`

| Field | Type | Required | Default |
|-------|------|----------|---------|
| `ussPath` | string | yes | — |
| `mode` | string | yes | — — `0755`, `u+x`… |
| `recursive` | boolean | no | `false` |

## Outputs

| Action | Outputs |
|--------|---------|
| `upload` MVS | `bytesTransferred`, `membersWritten` |
| `upload` USS | `bytesTransferred` |
| `download` | `bytesTransferred`, `localPath` |
| `create` | `created` (bool) |
| `delete` | `deleted` (bool) |
| `list` MVS | `datasets` or `members` (array), `count` |
| `list` USS | `entries` (array), `count` |
| `exists` MVS | `exists` (bool), `datasetType` |
| `exists` USS | `exists` (bool), `type` (`file`, `dir`, `symlink`) |
| `mkdir` | `created` (bool), `ussPath` |
| `chmod` | `success` (bool) |

## Typed errors

| Error | Description |
|-------|-------------|
| `ErrInvalidDatasetName` | Dataset/member name violating MVS rules. |
| `ErrDatasetNotFound` | Dataset or member not found. |
| `ErrDatasetExists` | `create` on an already-existing dataset. |
| `ErrMemberNotFound` | PDS member not found. |
| `ErrMemberNameCollision` | Two local files produce the same member name. |
| `ErrTransferFailed` | Upload or download failure. |
| `ErrUssPathNotFound` | USS path not found. |
| `ErrUssPermission` | USS operation denied by RACF. |

## Examples

**Upload a COBOL directory to a PDS**

```yaml
- uses: zos-datasets
  with:
    action: upload
    localPath: ./src/cobol
    dataset: USER.MYAPP.SRC
```

**Download a PDS member**

```yaml
- uses: zos-datasets
  with:
    action: download
    dataset: USER.MYAPP.SRC(CUSTOMER)
    localPath: ./review/CUSTOMER.cbl
```

**Conditional PDS allocation**

```yaml
- id: check-src
  uses: zos-datasets
  with:
    action: exists
    dataset: USER.MYAPP.SRC

- id: create-src
  if: ${{ steps.check-src.outputs.exists == false }}
  uses: zos-datasets
  with:
    action: create
    dataset: USER.MYAPP.SRC
    datasetType: pds
    recfm: FB
    lrecl: 80
    primary: 10
    secondary: 5
    dirblk: 20
```

**Deploy a script to USS**

```yaml
- uses: zos-datasets
  with:
    action: mkdir
    ussPath: /u/Z79863/deploy
    recursive: true

- uses: zos-datasets
  with:
    action: upload
    ussPath: /u/Z79863/deploy/run.sh
    localPath: src/run.sh

- uses: zos-datasets
  with:
    action: chmod
    ussPath: /u/Z79863/deploy/run.sh
    mode: "0755"
```

## Common errors

- **Member name collision** — `customer.cbl` and `customer.jcl` both produce `CUSTOMER`. Rename one of the files locally.
- **Corrupted encoding** — a load module uploaded without `binary: true` is mangled by EBCDIC conversion. Always `binary: true` for load modules.
- **`ErrDatasetExists`** — `create` fails if the dataset already exists. Use `exists` first or `continueOnError`.
